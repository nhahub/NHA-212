# YumifyPlus - Frontend

This is the frontend React application for YumifyPlus.

For complete setup instructions, project documentation, and API details, please refer to the main [README.md](../README.md) in the root directory.

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Frontend Tech Stack

- React 19
- Vite 7
- React Router v7
- Tailwind CSS
- Axios
- React Hot Toast

For more information, see the [main README](../README.md).
